## 🧭 What StarStrings does

StarStrings is a Windows app for applying MrKraken’s string replacements to add blueprint pools to contracts. It helps you update contract text without doing the work by hand. You use it to load your files, run the replacement set, and save the result.

StarStrings works on a Windows PC.

## 🚀 Getting started

1. Open the download page above.
2. Get the latest StarStrings file for Windows.
3. Save the file in a folder you can find, such as Downloads or Desktop.
4. If the file comes in a zip folder, extract it first.
